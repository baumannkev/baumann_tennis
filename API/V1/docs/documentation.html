Account

Up 
post /login
Login to your account (login)

Request body

email (required)
Body Parameter — Email of the user account

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Logged in successfully

500

Unable to retrieve email
Up 
post /signUp
Sign up for an account (signUpPost)

Request body

email (required)
Body Parameter — Email of the user account
password (required)
Body Parameter — Password of the account
phonenumber (required)
Body Parameter — Phone number of the user
name (required)
Body Parameter — Name of the user
address (required)
Body Parameter — Address of the user
insurance (optional)
Body Parameter — Insurance provider of the user
dateOfBirth (required)
Body Parameter — Date of birth of the user
sex (required)
Body Parameter — Sex of the user
allergies (optional)
Body Parameter — Allergies level of the user
emergency (required)
Body Parameter — Emergency contant number of the user
relationship (required)
Body Parameter — Relationship of the emergency contact number of the user

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

User account created

400

An error occurred. :(

500

Failed to associate the player and user
